Greenvale Pollution Optimization System

This project is a Java-based system that helps a city choose the best combination of pollution reduction projects while minimizing cost. Users can select mitigation options through an easy-to-use interface, and the system computes the optimal solution that meets all required pollution targets. The final output presents the selected projects, their implementation levels, and the total cost.

ICS Course Registration Planner

This project is a JavaFX-based application that assists students in planning their course registrations through an interactive and user-friendly interface. It allows students to manage courses within their degree program, view schedules on a weekly calendar, and access course details easily. The system also includes user authentication and provides clear feedback for invalid actions.

ITEA Furniture Order Form

This project is a responsive furniture ordering interface built using HTML and CSS, inspired by modern furniture brands like IKEA. It features a clean, user-friendly form that collects customer details, room information, and delivery preferences. The design focuses on clarity, layout consistency, and an intuitive user experience.
